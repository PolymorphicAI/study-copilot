# Study Copilot Drips Application Text

## Short description

Study Copilot is an AI-powered study buddy app that turns topics, notes, and PDFs into interactive micro-learning cards, quizzes, challenges, and progress flows.

## Types of work we will post

We would post scoped issues covering frontend UI, FastAPI backend improvements, AI prompt/output validation, PDF upload handling, API documentation, testing, accessibility, security, developer setup, and Stellar/Soroban research.

Examples include improving study card states, documenting AI prompts, testing API endpoints, improving PDF validation, creating accessibility checklists, researching Stellar learner identity, and designing Soroban-based learning badges or credentials.

## How this fits Stellar/Soroban

The current app is an AI learning tool. The Stellar/Soroban exploration focuses on learner identity, verifiable achievements, study badges, portable progress, and future learning reward flows.
