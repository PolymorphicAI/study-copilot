# Contributing to Study Copilot

Thank you for your interest in contributing to Study Copilot.

Study Copilot is an AI-powered study buddy app that turns topics, notes, and PDF materials into interactive micro-learning cards.

## How to Contribute

1. Check the Issues tab.
2. Choose an issue that matches your skill level.
3. Comment on the issue before starting.
4. Fork the repository.
5. Create a branch for your work.
6. Make focused changes.
7. Open a pull request with a clear description.

## Contribution Areas

- Frontend UI and accessibility
- FastAPI backend improvements
- AI prompt and output validation
- PDF upload and parsing improvements
- Testing and QA documentation
- Setup and developer experience
- Stellar/Soroban research for learner identity, credentials, and rewards

## Pull Request Checklist

Before opening a pull request, confirm:

- The PR matches the issue scope.
- Documentation is updated where needed.
- Tests or manual verification steps are included where appropriate.
- No API keys, secrets, `.env`, or private files are committed.
- The PR description clearly explains what changed.

## Complexity Guide

- Trivial = small documentation or setup task
- Medium = moderate feature, test, UI, or API task
- High = research-heavy, architecture, security, or integration task
